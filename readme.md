# Projeto de Réplica - Atlântico Avanti

## Contexto Acadêmico

Este projeto foi desenvolvido como atividade prática do curso de Fullstack, tendo como objetivo a reconstrução do website institucional do [Atlântico Avanti](https://www.atlanticoavanti.com.br). A proposta consistiu em replicar a estrutura, design e funcionalidades do site original, aplicando conhecimentos adquiridos ao longo do curso e explorando técnicas avançadas de desenvolvimento front-end.

### Visualização

Para poder visualizar o resultado, abra o arquivo **./public/index.html** utilizando algum servidor como o plugin **Live Server** do VSCode

## Objetivo do Projeto

O desafio proposto pelo curso visava não apenas reproduzir visualmente o website, mas também implementar soluções técnicas que demonstrassem compreensão profunda dos conceitos de desenvolvimento web moderno. Dessa forma, a equipe optou por ir além da simples replicação, incorporando práticas e padrões que elevassem a qualidade técnica do código produzido.

## Técnicas e Metodologias Implementadas

Durante o desenvolvimento, a equipe estabeleceu como meta a aplicação de diversas técnicas e boas práticas, incluindo:

### 1. Componentização com JavaScript
Implementação de uma arquitetura baseada em componentes reutilizáveis, permitindo maior modularidade do código e facilitando a manutenção futura.

### 2. Menus Dinâmicos com CSS
Desenvolvimento de sistemas de navegação responsivos utilizando técnicas avançadas de CSS, incluindo animações, transições e estados interativos, sem dependência excessiva de JavaScript para funcionalidades básicas de interface.

### 3. Image Replacement
Aplicação de técnicas de substituição de imagens para otimização de performance e acessibilidade, garantindo que elementos visuais sejam carregados de forma eficiente e que alternativas textuais estejam disponíveis para tecnologias assistivas.

### 4. Gradient Backgrounds
Utilização de gradientes CSS para criar fundos visualmente atraentes e modernos, reduzindo a necessidade de imagens pesadas e proporcionando maior flexibilidade no design responsivo.

### 5. Otimização para SEO
Considerando que o Atlântico Avanti é um programa de divulgação e formação tecnológica, dedicou-se especial atenção às práticas de Search Engine Optimization (SEO), incluindo:
- Estruturação semântica adequada do HTML
- Meta tags otimizadas
- Hierarquia apropriada de títulos (h1, h2, h3...)
- Atributos alt descritivos em imagens
- Performance e velocidade de carregamento
- Responsividade e mobile-first approach

## Equipe de Desenvolvimento

Este projeto foi desenvolvido colaborativamente por uma equipe de três estudantes, utilizando metodologias ágeis e controle de versão através do Git para garantir organização e rastreabilidade do código.

## Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo
- **CSS3**: Estilização e design responsivo
- **JavaScript**: Interatividade e componentização
- **Git**: Controle de versão e colaboração

## Considerações Finais

Este projeto representa não apenas um exercício de replicação técnica, mas uma oportunidade de aplicar conhecimentos teóricos em um contexto prático real, desenvolvendo habilidades essenciais para a atuação profissional em desenvolvimento web fullstack.
